# Khi bạn muốn tạm dừng Bot nhưng không muốn gỡ bỏ thì sao?

## Step 1 — Dừng dịch vụ gateway
```bash
openclaw gateway stop
```
Lệnh này dừng tiến trình chạy ngầm, Openclaw vẫn nằm trên máy của bạn

## Step 2 — Kiểm tra trạng thái
```bash
openclaw gateway status
```
Nên là "stopped", "not running". Nếu không phải, hãy thử `force stop`

## Step 3 — Ngăn tự khởi động
Nếu bạn cấu hình OpenClaw tự khởi động thì hãy dừng nó lại:

```bash
systemctl --user disable openclaw-gateway
```
## Nếu bạn muốn khởi động lại
```bash
openclaw gateway start
```
Lệnh này sẽ chỉ khởi động lại trong 1 phiên hoạt động

## Nếu bạn muốn bật lại "tự khởi động"
```bash
systemctl --user enable openclaw-gateway
systemctl --user start openclaw-gateway
```
