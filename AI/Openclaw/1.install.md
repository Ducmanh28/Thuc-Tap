# Cách cài đặt OpenClaw
Có 2 cách cài đặt OpenClaw
- Cài trực tiếp trên máy cá nhân
- Cài trên VPS

Ở đây tôi sẽ chọn cách 2
# Nguyên vật liệu cần thiết
Bạn sẽ cần có:
- Telegram Bot API(Tùy chọn) 
- Gemini API Key(Bắt buộc, tuy nhiên có thể dùng key LLM khác)
- Máy tính(cấu hình cơ bản)

# Thực hành
Bắt đầu quá trình cài đặt OC. Tôi sẽ cài đặt trên VPS chạy Ubuntu Server 24.04
## Cập nhật hệ thống
```
sudo apt update && apt upgrade -y
```
## Cài đặt
Truy cập trực tiếp vào:
```
https://openclaw.ai/
```
Copy câu lệnh cài đặt và paste vào thiết bị của bạn
```
curl -fsSL https://openclaw.ai/install.sh | bash
```
Sau khi chạy câu lệnh này, bạn sẽ đợi khoảng vài phút(Tùy tốc độ mạng) để OC cài đặt các môi trường cần thiết.

Sau khi hoàn tất, bạn sẽ nhận được câu hỏi liên quan đến khóa bảo mật. Ở đây chọn `yes`:
```
I understand this is personal-by-default and shared/multi-user use requires lock-down. Continue?
Yes/No
```
Tiếp đến chọn cách cài đặt: `Manual`
```
Onboarding mode:
QuickStart
Manual
```
Tiếp theo là chọn nơi cài đặt: `Local gateway`
```
What do you want to set up?
Local gateway (This machine)
Remote gateway (info-only)
```
Tiếp đến là nơi làm việc mặc định: Enter mặc định
```
Workspace directory
/root/.openclaw/workspace
```
Tiếp theo là chọn LLM: Ở đây tôi sử dụng `Google` gemini
```
Model/auth provider
Google
```
Chọn phương thức xác thực: 
```
Google auth method:
Google API Key
```
Chọn cách mà bạn cung cấp Key: 
```
How do you want to provide this API key
Paste API key now
```
Nhập API Key:
```
Enter Google API key
Ajasjfjasfii184711bf8b83fb93un
```
Tiếp đến là chọn model. Ở đây tôi cung cấp cho bạn 1 file `test.py` để bạn kiểm tra xem API key của bạn có thể sử dụng model nào.
```
Default model
google/gemini-flash-lite-lastest
```
Tiếp đến sẽ chọn cổng đầu ra:
```
Gateway Port
18789
```
