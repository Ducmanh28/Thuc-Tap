# Job
Job là một Kubernetes Controller chuyên trách quản lý các tiến trình xử lý hàng loạt (batch processing) có tính chất hữu hạn. Khác với Deployment/DaemonSet (duy trì trạng thái Running liên tục), Job Controller cấp phát một hoặc nhiều Pods, giám sát vòng đời của chúng cho đến khi quá trình thực thi kết thúc thành công với Exit Code 0 (chuyển sang trạng thái Completed).

# Ứng dụng (Use Cases):
- Database Migrations (cập nhật schema database).
- Batch Data Processing (xử lý khối lượng dữ liệu lớn theo chu kỳ hoặc sự kiện).
- One-off Scripts (thực thi script dọn dẹp, khởi tạo môi trường).
- Data Backups/Snapshots.

# Thông số:
- completions: Số lần Pod cần kết thúc thành công.
- parallelism: Số lượng Pod được phép chạy đồng thời.
- backoffLimit: Số lần thử lại (retries) tối đa khi Pod thất bại (Exit Code != 0) trước khi đánh dấu toàn bộ Job là Failed.
- restartPolicy: Bắt buộc thiết lập là Never hoặc OnFailure (không chấp nhận Always).

# Job101
Ví dụ: Khởi tạo một Job tính toán hằng số Pi đến 2000 chữ số thập phân sử dụng Perl, sau đó tự động kết thúc

```YAML
apiVersion: batch/v1
kind: Job
metadata:
  name: pi-calculator
spec:
  completions: 1      # Yêu cầu 1 Pod thực thi thành công
  backoffLimit: 4     # Cho phép retry tối đa 4 lần nếu lỗi
  template:
    spec:
      containers:
      - name: pi
        image: perl:5.34.0
        command: ["perl",  "-Mbignum=bpi", "-wle", "print bpi(2000)"]
      restartPolicy: Never # Không tự động restart container, tạo Pod mới nếu lỗi
```
Tạo job:
```bash
root@k8s-master:/home/ducmanh/k8s# kubectl apply -f test.yaml
job.batch/pi-calculator created
```
Kiểm tra vòng đời Pod:
```
root@k8s-master:/home/ducmanh/k8s# kubectl get pods -l job-name=pi-calculator -w
NAME                  READY   STATUS              RESTARTS   AGE
pi-calculator-7k8v2   0/1     Pending             0          0s
pi-calculator-7k8v2   0/1     ContainerCreating   0          1s
pi-calculator-7k8v2   1/1     Running             0          4s
pi-calculator-7k8v2   0/1     Completed           0          12s
```
Khi tiến trình perl bên trong container hoàn tất việc tính toán và thoát với Exit Code 0, kubelet báo cáo trạng thái về Control Plane. Job Controller đánh dấu Pod là Completed. Pod KHÔNG bị xóa đi (để giữ lại logs) nhưng cũng KHÔNG tiêu thụ thêm tài nguyên CPU/RAM

Kiểm tra jobs:
```
root@k8s-master:/home/ducmanh/k8s# kubectl get jobs
NAME            COMPLETIONS   DURATION   AGE
pi-calculator   1/1           56s        118s
```
Cột COMPLETIONS 1/1 xác nhận Job đã đạt đủ số lượng thực thi thành công yêu cầu. DURATION 12s là thời gian thực tế tính từ lúc Pod bắt đầu chạy đến khi kết thúc
