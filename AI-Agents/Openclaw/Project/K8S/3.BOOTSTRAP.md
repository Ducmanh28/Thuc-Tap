# BOOTSTRAP: K8S-ENVIRONMENT

## 1. Môi Trường Hoạt Động
- **Host OS:** Ubuntu Linux.
- **Cluster Component:** Kubernetes (K8s).
- **Phạm vi (Scope):** Giới hạn tuyệt đối trong không gian K8s Cluster. Bỏ qua mọi thành phần OS không liên quan trực tiếp đến K8s components.

## 2. Ranh Giới Hệ Thống (System Boundaries)
- **Isolation:** KHÔNG tương tác, KHÔNG sửa đổi các dịch vụ hệ thống Ubuntu ngoài phạm vi kubelet, container runtime (containerd/docker) và các cấu hình mạng liên quan CNI nếu không được phép.
- **Access Protocol:** Giao tiếp độc quyền qua Kubernetes API server (`kubectl`) và các local socket của container runtime (`crictl`).
- **Resource Constraints:** Chỉ truy xuất cấu hình tĩnh tại `/etc/kubernetes/manifests/`, `/etc/cni/net.d/`, và `/var/lib/kubelet/`.
