# Cách cài đặt OpenClaw
Có 2 cách cài đặt OpenClaw
- Cài trực tiếp trên máy cá nhân
- Cài trên VPS

Ở đây tôi sẽ chọn cách 2

# Thực hành
Bắt đầu quá trình cài đặt OC. Tôi sẽ cài đặt trên VPS chạy Ubuntu Server 24.04
## Cập nhật hệ thống
```
sudo apt update && apt upgrade -y
```
## Cài đặt
Truy cập trực tiếp vào:
```
https://openclaw.ai/
```
Copy câu lệnh cài đặt và paste vào thiết bị của bạn
```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```
Sau khi chạy câu lệnh này, bạn sẽ đợi khoảng vài phút(Tùy tốc độ mạng) để OC cài đặt các môi trường cần thiết.

```
🦞 OpenClaw Installer
  I speak fluent bash, mild sarcasm, and aggressive tab-completion energy.
```

Sau khi hoàn tất, bạn sẽ nhận được câu hỏi liên quan đến khóa bảo mật. Ở đây chọn: `yes`
```
 I understand this is personal-by-default and shared/multi-user use requires lock-down. Continue?
│  ● Yes / ○ No
└
```
Tiếp đến chọn cách cài đặt: `Manual`
```
◆  Setup mode
│  ○ QuickStart
│  ● Manual (Configure port, network, Tailscale, and auth options.)
```
Tiếp theo là chọn nơi cài đặt: `Local gateway`
```
◆  What do you want to set up?
│  ● Local gateway (this machine) (No gateway detected (ws://127.0.0.1:18789))
│  ○ Remote gateway (info-only)
```
Tiếp đến là nơi làm việc mặc định: Enter mặc định
```
◆  Workspace directory
│  /root/.openclaw/workspace█
```
