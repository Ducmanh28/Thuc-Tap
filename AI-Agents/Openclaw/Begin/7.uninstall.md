# Gỡ cài đặt Openclaw
Gỡ cài đặt hoàn toàn Openclaw
## Để openclaw tự gỡ
```bash
openclaw uninstall
```
## Dừng gateway
```bash
openclaw gateway stop
```

## Xóa gói npm
```bash
npm uninstall -g openclaw
```
Xóa CLI 

## Xóa file config
```bash
rm -rf ~/.openclaw
```
Xóa tất cả cấu hình

## Xóa nơi làm việc của Openclaw (tùy chọn)
```bash
rm -rf ~/clawd
```
Đây là nơi chứa các ghi chú, dữ liệu, tài liệu của bạn. Xóa nếu không cần thiết

## Thu hồi API Key
Truy cập vào giao diện quản lý API Keys của bạn và thực hiện thu hồi API Key
