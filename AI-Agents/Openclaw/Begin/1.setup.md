# Nguyên vật liệu cần thiết
Bạn sẽ cần có:
- Telegram Bot API(Tùy chọn) 
- Gemini API Key(Bắt buộc, tuy nhiên có thể dùng key LLM khác)
- Máy tính(cấu hình cơ bản)

# Thông số cài đặt:
- Sử dụng Telegram Bot 
- Dùng LLM: gemini-3-flash-lite
- Cài đặt trên thiết bị: Ubuntu Server 24.04
  - RAM: 2GB
  - DISK: 50GB
  - CPU: 2GHz
  - Network: Ổn định, khá nhanh
- Sử dụng Laptop SSH vào Server để cài đặt
